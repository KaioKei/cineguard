# Cineguard

A nice tool to manage your favorite movies or those you wanna watch

## Quickstart

Build and start :

```sh
make build
./cineguard serve
# ...
# [GIN-debug] Listening and serving HTTP on 127.0.0.1:8080
```

Check status :

```sh
curl localhost:8080/api/v1/health
```

## Build

```sh
make build
```
